# Lion Checkout

Web Component for checkout flow.

## Prerequisites

* Node.js 14 or higher - Download & Install Node.js and the npm package manager.

## Installing

Open a terminal and go to the root of the project.
Run `npm install` to install all dependencies.

## Starting application
Open a terminal and cd to the root of the project. Start the application by running `npm run start:build`.

## Usage
Go to `http://localhost:8000` to see the Web Component.

## Developing
Open a terminal and cd to the root of the project. Start the application by running `npm run dev`.
The application will restart automatically on file change.

## Documentation
Open your browser and go to [Link to Docs] to see the documentation.

## Running the tests

Run `npm test` to run the unit tests (not implemented yet).
